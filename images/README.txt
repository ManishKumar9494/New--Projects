============================================================
KW New Projects — Images Folder
============================================================

Drop the following image files into this /images/ folder using the
EXACT filenames below. Until you add them, the website automatically
shows a tasteful navy→copper gradient placeholder (with a "KW" monogram),
so the layout never looks broken.

Recommended format: optimised .jpg (or .webp renamed to .jpg).
Keep each file under ~300 KB for fast loading. Use real, high-quality
architectural / interior / amenity photography or artist impressions.
